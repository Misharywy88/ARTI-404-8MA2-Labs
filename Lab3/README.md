# 🖼️ ARTI 404: Lab 3 - Image Transformations
This repository contains the implementation of **Geometric** and **Intensity** transformations using Python and OpenCV.

## 🛠️ Tasks Performed:
### 📐 Geometric Transformations
1. **Scaling:** Increased image size using Cubic Interpolation.
2. **Rotation:** Rotated the image by 120° clockwise.
3. **Shearing:** Applied horizontal shear mapping.

### 💡 Intensity Transformations
1. **Negative:** Inverted pixel intensities.
2. **Log Transform:** Expanded dark pixel values.
3. **Power Law (Gamma):** Enhanced contrast using $\gamma = 0.5$.

## 📦 Requirements
- Python 3
- OpenCV (`cv2`)
- NumPy
- Matplotlib
